# PythonFundamentos
Repositório criado para armazenar exercícios extraídos do livro: ASCENCIO, Ana Fernanda Gomes; CAMPOS, Edilene Aparecida Veneruchi de. Fundamentos da programação de computadores: algoritmos, PASCAL, C/C++ (padrão ANSI) e JAVA. 3. ed. São Paulo: Pearson, 2012. E-book. Os exercícios foram feitos utlizando a linguagem Python.
